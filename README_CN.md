## PullRequest需求

1. Bug修复
2. 性能优化
3. 语言翻译
4. 游戏内容更新（新英雄、种族/职业和技能改动）

2. 阵容搭配支持生成图片分享到社交网络
3. 支持物品及合成路线查询
4. 支持查看英雄属性数据和技能详情数据
